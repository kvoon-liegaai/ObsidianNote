目标：不访问原始服务器，就满足客户请求

- 用户设置浏览器：通过缓存访问 Web
- 浏览器将所有 HTTP 请求发送给缓存

![[img/Pasted image 20220705154707.png]]

缓存即是客户端又是服务端

## <span style="color:red;font-weight:bold">为什么要使用 web 缓存？</span>

- 客户端：请求时间降低
- 链路：大大减少一个机构内部网络与 Internet 接入链路上的流量
- 服务商：可是使较弱的 ICP（电信运营商） 也能够提供有效的内容
