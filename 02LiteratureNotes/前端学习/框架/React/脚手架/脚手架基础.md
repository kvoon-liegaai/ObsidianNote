## 脚手架基础

- rcc（React class component） 快速创建React模板代码片段  
	- rfc( React function component )  
- manifest.json 安卓引用加壳  
- robots.txt 爬虫规则  
- App.test.js 用于App的测试，几乎不用  
- <React.StrictMode> 用于检测App组件及其子组件的写法是否合理  
- setupTest.js 用于页面的整体测试
