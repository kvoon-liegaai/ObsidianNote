## 媒体组件

`image`

image 组件默认看宽度约 300 px，高度约 240 px

属性

- `mode`

| 值          | 说明                             |
| ----------- | -------------------------------- |
| scaleToFill | 拉伸以填充                       |
| aspectFit   | 长边能显示出（画面留有空白）     |
| aspectFill  | 保证短边能显示出来（画面被裁剪） |
| widthFix    | 宽度不变，高度自适应             |
| heightFix   | 高度不变，宽度自适应             |
