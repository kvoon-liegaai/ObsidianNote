# 柯里化收集state

[[../../../../../三件套+/JS高级/柯里化/柯里化|柯里化定义]]

通过柯里化，我们可以**将要收集数据名缓存起来**

```jsx
class Person extends React.component{
	render(){
		(
			<form>
				<input onChange={ this.saveData("username") } />
				<input onChange={ this.saveData("address") } />
			</form>
		)
	}
	saveData = (dataType)=>{
		return (event)=>{
			setState({[dataType]:event.target.value});
		}
	}
}
```
