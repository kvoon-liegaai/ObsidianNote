## scoped
父组件

```vue
<template>
	<div>
		<h2>我是父组件</h2>
		<Child></Child>
	</div>
</template>

<style scoped>
	h2{
		color:red;
	}
</style>
```

```vue
<template>
	<div>
		<h2>我是子组件</h2>
	</div>
</template>

<style scoped>
	h2{
		color:orange;
	}
</style>
```

父组件使用 `scoped` 后，样式很难应用到子组件身上
