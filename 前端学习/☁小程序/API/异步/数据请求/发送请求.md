## 发送请求

使用 `wx.request` API

注意: 该 API 并不支持 [[异步 API Promise 风格#异步 API Promise 风格|promise 风格编程]]

get 请求

```ts
ws.request({
	url:"https://www.escook.cn/api/get",
	method:"GET",
	data:{
		name:"kwongliegaai",
		age:21
	},
	success:(res) => {
		const {data} = res; // data 为服务器返回的真实数据
		console.log(data);
	}
})
```


post 请求

```ts
wx.request({
	url:"https://www.escook.cn/api/post",
	method:"POST",

		name:"kwongliegaai",
		age:21

```
