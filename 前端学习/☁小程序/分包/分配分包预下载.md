## 分配分包预下载

配置 `app.json`  

`"preloadRule"` 字段下的 页面路径字段

| 属性     | 类型     | 取值        | 含义                                   |
| -------- | -------- | ----------- | -------------------------------------- |
| 网络     | string   | all/wifi    | 指定网络条件进行预下载                 |
| packages | string[] | root/name值 | 通过 root 或者 name 指定预下载那些分包 |

```json
{
	"preloadRule":{// 分包预下载的规则
		"pages/contact/contact":{// 触发分包预下载的页面路径
		"network":"all",
		"packages":["pkgA"]
		}
	}
}
```

### 限制

同一个分包中的页面享有共同的预下载大小限额 2M，
