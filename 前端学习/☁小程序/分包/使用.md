## 使用

![[img/Pasted image 20220830143752.png]]

对应 `app.json` 

配置 `subPackages`

```json
{
	"pages":[
		"pages/index",
		"pages/logs"
	],
	"subPackages":[
		{
			"root":"packageA",
			"name":"p1"
			"pages":[
				"pages/cat",
				"pages/dog"
			]
		},
		{
			"root":"packageB",
			"name":"p2"
			"pages":[
				"pages/apple",
				"pages/banana"
			]
		}
	]
}
```

保存后将自动生成文件
